# Crypto Data Analyzer

Crypto Data Analyzer is a web application built using Streamlit that provides interactive charts and technical analysis for various cryptocurrencies. Users can select different time intervals to view the data and download the analyzed data as an Excel file.

## Features

- Interactive candlestick charts
- Technical indicators data
- Multi-language support (English and Arabic)
- Download data as Excel file

## Requirements

- Streamlit
- Pandas
- Requests
- Plotly
- TA-Lib
- Openpyxl

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/crypto-data-analyzer.git
cd crypto-data-analyzer


Install the required packages:
pip install -r requirements.txt

Usage
To run the application, use the following command:
streamlit run app.py

Deployment
The application can be deployed using Streamlit Cloud for free. Follow these steps:

Push your code to a GitHub repository.
Visit Streamlit Cloud and sign in with your GitHub account.
Click on "New app" and select your repository and branch.
Configure the app and click "Deploy".
